CAE Laminate Solver

January 2026

A sleek mobile app interface used to calculate laminate stiffness.It acts as a specialized computational engine used to analyze the structural behavior of composite materials. Unlike isotropic materials (like steel or aluminum), composites are made of multiple layers (plies) with different orientations, making their physics significantly more complex.

Data Structures & Algorithms

This focus area involves the rigorous study and implementation of efficient algorithms and complex data structures to solve computational problems. The work is grounded in the application of discrete mathematics—including graph theory, combinatorics, and logic—to optimize time and space complexity. Key areas of exploration include memory management, sorting and searching efficiency, and the implementation of advanced structures like heaps, trees, and hash maps in C++. I have applied these principles to build optimized solutions for real-world scenarios, such as Dijkstra’s algorithm for pathfinding and Prim’s/Kruskal’s for network optimization. Within my peer groups at VIT, I often take the lead on algorithm design and complexity analysis, ensuring that our projects are not only functional but also scalable and performant.

Structured and Object Oriented Programming

This discipline is concerned with the transition from procedural programming to a modular or blueprint-based approach to software development. Through the mastery of the fundamental pillars of OOPS—Encapsulation, Abstraction, Inheritance, and Polymorphism—I have learned to develop systems not only fault-tolerant but also highly reusable and maintainable. This subject is a deep dive into the ability to package data and behavior into "objects," making the modeling of real-world systems much more intuitive. Throughout this course, I have been primarily concerned with the application of these programming concepts in C++ and Python programming languages.
